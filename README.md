# Student Admissions & Uniform Management System

A Flask-based web application that streamlines the school admissions process from enquiry to enrolment while providing a dedicated uniform management portal.

## Features

- Student admissions management
- Multi-stage admissions workflow
- Assessment scheduling and scoring
- Student activity tracking
- Printable student records
- Live class capacity dashboard
- Separate Uniform Management portal
- Uniform collection tracking
- Secure role-based authentication

## Tech Stack

- Python
- Flask
- SQLite
- HTML
- CSS
- Jinja2

## Running Locally

```bash
git clone https://github.com/yourusername/student-admissions.git
cd student-admissions
pip install -r requirements.txt
python app.py
```

Open:

```
http://localhost:5000
```
